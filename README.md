# HotelesReactNative

# 1 install dependencies
```bash 
    yarn install
```

# 2 configuration Api
editar el archivo de configuracion para editar el host del Api <br/>
./src/config.ts <br/>

```javascript
export const config = {
    hostUrl: "http://....."
}
```


# 3 running app
```bash 
npm start
npm run build 
npm run run:android 
```
